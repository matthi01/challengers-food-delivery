<h1>Challenger's Food Delivery</h1>

<p>Requirements:
<ul>
<li>The app should display 2 pages:</li>
<ul><li>Restaurants & Menu</li><li>My Orders</li></ul>
<li>On the “Restaurants & Menu” page, the user should be able to select a restaurant and see their
menu items with associated prices. You will need to retrieve the data for each restaurant from
this endpoint:</li>
<ul><li>https://huddolapi-next.herokuapp.com/v1/challenge</li></ul>
</ul></p>

<h3>Running Locally</h3>
<p>To run locally:</p>
<ul>
    <li>Download the code</li>
    <li>Open a terminal window and move into the project directory</li>
    <li>Run
        <code>npm install</code>, this will install any required dependencies to run the react app</li>
        <li>Run <code>npm start</code>, this will launch the app</li>
</ul>
